# 설치 가이드

## 📥 다운로드

### 방법 1: ZIP 다운로드 (권장)

1. 이 페이지 상단의 초록색 **`< > Code`** 버튼 클릭
2. **`Download ZIP`** 클릭
3. 다운받은 ZIP 파일 압축 해제
4. 압축 해제된 폴더를 원하는 위치에 이동 (예: `C:\testreview` 또는 바탕화면)

> ⚠️ **주의**: 폴더 경로에 한글이 없는 곳에 저장하는 것을 권장합니다.

### 방법 2: git clone

git이 설치되어 있다면:
```bash
git clone https://github.com/anzi/testreview.git
cd testreview
```

## 🖥️ Windows 설치

### 1단계: 설치 스크립트 실행

1. 압축 해제한 폴더에서 **`install.bat`** 파일 찾기
2. **우클릭 → 관리자로 실행** 클릭

   > 일반 더블클릭이 아닌 **관리자로 실행**이어야 합니다.

3. "이 앱이 변경하도록 허용하시겠어요?" → **예** 클릭

설치 스크립트가 자동으로:
- ✅ Git 설치 (없을 경우)
- ✅ Python 3.11 설치 (없을 경우)
- ✅ Node.js 설치 (없을 경우)
- ✅ 필요한 패키지 설치
- ✅ 데이터베이스 초기화
- ✅ 바탕화면에 실행 파일 생성

> 인터넷 연결 상태에 따라 **5~15분** 소요될 수 있습니다.

### 2단계: 실행

바탕화면의 **`채점기준 정제 도구`** 파일을 더블클릭하면 브라우저가 자동으로 열립니다.

### 3단계: 초기 설정

처음 실행 시 관리자 계정 설정 화면이 나타납니다. 아이디와 비밀번호를 설정하세요.

## 🍎 Mac 설치

### 1단계: 설치 스크립트 실행

터미널을 열고 압축 해제한 폴더로 이동 후 실행:

```bash
cd ~/Downloads/testreview  # 실제 경로로 변경
bash install.sh
```

또는 Finder에서 폴더를 연 후, 터미널에 `bash ` 입력하고 `install.sh` 파일을 드래그&드롭 후 엔터.

설치 스크립트가 자동으로:
- ✅ Git 확인 (없으면 설치 안내)
- ✅ Homebrew 설치 (없을 경우)
- ✅ Python 3.11 설치 (없을 경우)
- ✅ Node.js 설치 (없을 경우)
- ✅ 필요한 패키지 설치

### 2단계: 실행

**`start.command`** 파일을 더블클릭하면 브라우저가 자동으로 열립니다.

> 처음 실행 시 "확인되지 않은 개발자" 경고가 뜰 수 있습니다.
> 이 경우: Finder에서 파일 **우클릭 → 열기 → 열기** 를 선택하면 이후부터는 그냥 더블클릭으로 실행됩니다.

### 3단계: 초기 설정

처음 실행 시 관리자 계정 설정 화면이 나타납니다.

## 🔑 Gemini API 키 설정

설치 후 앱에서 Gemini API 키를 등록해야 AI 기능을 사용할 수 있습니다.

1. [Google AI Studio](https://aistudio.google.com) 접속 (구글 계정 로그인)
2. 왼쪽 메뉴 **Get API Key** → **Create API key** 클릭
3. 생성된 키 복사
4. 앱 로그인 후 **설정 → Gemini API 키 설정** 에 붙여넣기
5. **키 테스트** 버튼으로 정상 동작 확인

---

## 🔄 업데이트

앱 내에서 바로 업데이트할 수 있습니다:

1. 앱 로그인
2. 우측 상단 이름 클릭 → **API 키 설정** 이동
3. 하단 **업데이트 확인** 버튼 클릭
4. 새 버전이 있으면 **지금 업데이트** 클릭
5. 완료 후 앱 재시작 (start.bat 또는 start.command 다시 실행)

> 데이터(시험, 학생 답안, 채점 결과)는 업데이트 시 유지됩니다.

## ❓ 자주 묻는 질문

**Q. 인터넷이 없어도 사용할 수 있나요?**
A. AI 기능(채점기준 추출, 답안 OCR, 채점기준 정제)은 Gemini API를 사용하므로 인터넷이 필요합니다.

**Q. 학생 답안이 외부로 유출되지 않나요?**
A. 데이터는 Google의 Gemini API로만 전송됩니다. 별도 서버가 없으며, 본인의 Google AI API 키 계정 아래에서만 처리됩니다. 소스코드는 GitHub에 전체 공개되어 있어 확인 가능합니다.

**Q. 여러 명이 같이 쓸 수 있나요?**
A. 현재는 1인 로컬 앱입니다. 각자 설치해서 사용하세요.

**Q. 시험지를 올리면 문제가 유출되지 않나요?**
A. Gemini API의 데이터 보호 정책에 따라 처리됩니다. 민감한 시험이라면 채점기준 초안 생성 기능 대신 채점기준표를 직접 작성해 올리는 것을 권장합니다.

**Q. install.bat 실행 시 오류가 나요.**
A. 반드시 **관리자로 실행**해야 합니다. 일반 더블클릭으로는 권한 오류가 발생할 수 있습니다.
