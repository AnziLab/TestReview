# 설치 가이드

## 📥 다운로드

GitHub 저장소에서 **설치 파일 1개만** 다운로드하면 됩니다.
설치 스크립트가 Git으로 전체 프로젝트를 자동 클론하므로, 이후 업데이트도 자동입니다.

| OS | 다운로드 파일 |
|----|-------------|
| Mac | [`install.command`](https://raw.githubusercontent.com/anzilab/testreview/main/install.command) |
| Windows | [`install.bat`](https://raw.githubusercontent.com/anzilab/testreview/main/install.bat) |

> 위 링크를 우클릭 → **다른 이름으로 링크 저장** 으로 다운로드하세요.

---

## 🍎 Mac 설치

### 1단계: 설치

1. 위에서 다운받은 **`install.command`** 파일을 더블클릭
2. "확인되지 않은 개발자" 경고가 나오면: **우클릭 → 열기 → 열기**
3. 자동으로 설치가 진행됩니다 (5~15분 소요)

설치 스크립트가 자동으로:
- ✅ Git 설치 (Xcode Command Line Tools)
- ✅ Homebrew 설치 (없을 경우)
- ✅ Python 설치 (없을 경우)
- ✅ Node.js 설치 (없을 경우)
- ✅ 프로젝트 다운로드 (`~/TestReview`에 Git 클론)
- ✅ 필요한 패키지 설치
- ✅ 데이터베이스 초기화

### 2단계: 실행

`~/TestReview` 폴더의 **`start.command`** 파일을 더블클릭하면 브라우저가 자동으로 열립니다.

### 3단계: 초기 설정

처음 실행 시 관리자 계정 설정 화면이 나타납니다.

## 🖥️ Windows 설치

### 1단계: 설치

1. 위에서 다운받은 **`install.bat`** 파일을 **우클릭 → 관리자로 실행**
2. "이 앱이 변경하도록 허용하시겠어요?" → **예** 클릭
3. 자동으로 설치가 진행됩니다 (5~15분 소요)

   > 일반 더블클릭이 아닌 **관리자로 실행**이어야 합니다.

설치 스크립트가 자동으로:
- ✅ Git 설치 (winget 사용)
- ✅ Python 3.11 설치 (없을 경우)
- ✅ Node.js 설치 (없을 경우)
- ✅ 프로젝트 다운로드 (`%USERPROFILE%\TestReview`에 Git 클론)
- ✅ 필요한 패키지 설치
- ✅ 데이터베이스 초기화
- ✅ 바탕화면에 실행 파일 생성

### 2단계: 실행

바탕화면의 **`TestReview`** 파일을 더블클릭하면 브라우저가 자동으로 열립니다.

### 3단계: 초기 설정

처음 실행 시 관리자 계정 설정 화면이 나타납니다. 아이디와 비밀번호를 설정하세요.

## 🔑 Gemini API 키 설정

설치 후 앱에서 Gemini API 키를 등록해야 AI 기능을 사용할 수 있습니다.

1. [Google AI Studio](https://aistudio.google.com) 접속 (구글 계정 로그인)
2. 왼쪽 메뉴 **Get API Key** → **Create API key** 클릭
3. 생성된 키 복사
4. 앱 로그인 후 **설정 → Gemini API 키 설정** 에 붙여넣기
5. **키 테스트** 버튼으로 정상 동작 확인

## 🔄 업데이트

Git으로 설치되었기 때문에 업데이트가 자동으로 처리됩니다.

### 자동 업데이트

앱을 실행(start.command / start.bat)할 때마다 자동으로 최신 버전을 확인하고 업데이트합니다.
별도로 할 일이 없습니다.

### 수동 업데이트

앱 내에서도 업데이트할 수 있습니다:

1. 우측 상단 이름 클릭 → **API 키 설정**
2. 하단 **업데이트 확인** 버튼 클릭

> 데이터(시험, 학생 답안, 채점 결과)는 업데이트 시 유지됩니다.

## ❓ 자주 묻는 질문

**Q. 인터넷이 없어도 사용할 수 있나요?**
A. AI 기능(채점기준 추출, 답안 OCR, 채점기준 정제)은 Gemini API를 사용하므로 인터넷이 필요합니다.

**Q. 학생 답안이 외부로 유출되지 않나요?**
A. 데이터는 Google의 Gemini API로만 전송됩니다. 별도 서버가 없으며, 본인의 Google AI API 키 계정 아래에서만 처리됩니다. 소스코드는 GitHub에 전체 공개되어 있어 확인 가능합니다.

**Q. 여러 명이 같이 쓸 수 있나요?**
A. 현재는 1인 로컬 앱입니다. 각자 설치해서 사용하세요.

**Q. 시험지를 올리면 문제가 유출되지 않나요?**
A. Gemini API의 데이터 보호 정책에 따라 처리됩니다. 민감한 시험이라면 채점기준 초안 생성 기능 대신 채점기준표를 직접 작성해 올리는 것을 권장합니다.

**Q. install.bat 실행 시 오류가 나요.**
A. 반드시 **관리자로 실행**해야 합니다. 일반 더블클릭으로는 권한 오류가 발생할 수 있습니다.
